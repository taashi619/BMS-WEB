# BMS Admin Web

Administrative web application for the BMS platform, built to work with the shared backend API and provide a browser-based interface for management tasks.

## Live Demo

[Open the deployed app](https://your-admin-web-url.onrender.com)

## Project Summary

This repository contains the admin frontend for the BMS project. It is the web-based client that connects to the backend API and allows users to interact with the system through a browser interface.

This repository is part of a broader full-stack portfolio project that includes a backend API and a mobile client in separate repositories.

## What This Demonstrates

- Building a frontend client that consumes an external API
- Connecting a React-based interface to a backend service
- Handling environment-based API configuration
- Structuring a deployable frontend project using a modern JavaScript stack

## Tech Stack

- React
- Vite
- Axios
- Frontend routing
- API-driven UI

## Key Features

- Authentication-connected frontend flows
- API-driven user interface behavior
- Browser-based management experience
- Environment-based configuration
- Structured frontend architecture

## Running Locally

1. Install dependencies with `npm install`
2. Start the local development server with `npm run dev`
3. Open the local browser URL shown in the terminal
